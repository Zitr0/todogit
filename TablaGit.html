Comandos principales

Comando Detalle
git init Ubicado dentro de un folder, se inicia el directorio de trabajo
git status Ver el estado del folder
git add %nombrearchivo% Agregar un archivo al staging
git rm %nombrearchivo% Remover un archivo del staging
git rm --cached %nombrearchivo% Removerlo completamente del staging
git rm --force %nombrearchivo% Removerlo completamente del staging y el disco
git commit -m "Mensaje especifico" Enviarlo al master
git config git config --list git config --list --show-origin Ver la configuración de git
git config --global user.name "NombreUsuario"
git config --global user.email "Correo@mail.com" 		Agregar usuario y email
git log %nombrearchivo% Ver cambios o historia de un archivo
git show %nombrearchivo% Mostrar más información de las modificaciones del archivo
git reset -alfanumerico- --hard o --soft Devolver un archivo a un punto en la historia
Tener mucho cuidado porque este comando borra todos los cambios adicionales que se hicieron
git log --stat Permite visualizar los bits por commit
git checkout -alfanumerico- %nombrearchivo% Permite visualizar un archivo con la información de ese momento
se debe tener cuidado de no hacer commit en ese momento,
porque se perderia la información
git checkout master %nombrearchivo% si quiero volver al archivo
git checkout %nombrerama% Pararse en la rama
git merge %nombrerama% Fusionar lo realizado y dejarlo en la rama seleccionada
tener presente donde se esta ubicado en la rama
alias %nombre%="git log --all --graph --decorate --oneline" anteponiendo la palabra alias, se crea una forma rápida de llamar el comando
. ~/.bashrc Para agregar el alias de forma permanente, ubicados sobre la raiz ~
se crea el archivo nano .bashrc y dentro se le pone el comando
luego se guarda el archivo.
Finalmente se pone el comando.
